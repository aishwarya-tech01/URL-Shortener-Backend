# 📚 Library Management System

A lightweight, fully interactive College Library Management web application built using Python, Flask, and SQLite3.

## 🚀 Features
* **Live Inventory Tracking:** Displays total and available book copies dynamically.
* **Role-Based Simulation:** Toggle seamlessly between Student and Librarian interfaces.
* **Interactive Workflows:** Click "Request Loan" to borrow books or "Mark Returned" as a librarian to manage returns.
* **SQLite Backend:** Automatic database generation and tracking.
